Kinect22Scratch
===============

Kinect2 -> Scratch


## Requirements

 * Visual Studio 2012
 * Kinect for Windows v2 Developer Preview
 * Kinect for Windows SDK v2 Developer Preview 1311
 * [Scratch 1.4](http://info.scratch.mit.edu/ja/Scratch_1.4_Download)
